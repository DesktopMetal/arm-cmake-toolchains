## K64F Simple Blinking

Simple LED blinking on [FRDM-K64F][1] which is based on MK64FN1M0VLL12 IC

**Tested compilers**

- [ARM GCC 9.3.1][armgcc] - (_gcc-arm-none-eabi-9-2020-q2-major_)
- [Clang 8.0.0][clang]
- [Clang 9.0.1][clang]
- [Clang 10.0.0][clang]
